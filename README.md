# Student Management System

This project is a Spring Bootâ€“based application designed to manage student information efficiently.  
It provides basic operations such as adding, updating, viewing, and removing student records.

## Tech Stack
- Java  
- Spring Boot  
- Maven  
- REST APIs  
- Database support (MySQL)

## Features
- Create new student records  
- Retrieve student details  
- Modify existing student information  
- Delete student data  
- RESTful API structure

## How to Run the Application
1. Clone the repository from GitHub  
2. Open the project in any Java IDE (IntelliJ, Eclipse, VS Code)  
3. Configure the database in `application.properties`  
4. Run the `DemoApplication` class  
5. Access the APIs using Postman or a browser

## Purpose
This project was developed for learning and academic purposes to demonstrate Spring Boot fundamentals and backend development concepts.
